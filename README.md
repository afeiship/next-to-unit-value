# next-to-unit-value
> Get unit with value from string

## install:
```bash
npm install -S afeiship/next-to-unit-value --registry=https://registry.npm.taobao.org
```

## usage:
```js
import toUnitValue from 'next-to-unit-value';

const [_, value, unit ]=nx.toUnitValue('1.23rem');


// value: '1.23'
// unit: 'rem'
```
